📑코드의 반복을 피하자❗

❓클래스❓
- 반복되는 데이터를 묶는다
- 묶은 데이터를 사용하는 기능을 만들때 사용

❓생성자❓
- 묶인 데이터를 초기화 할때 사용
- 생성자에 포함되어 있지 않는 필드(데이터)는 null
- 생성자에 기능 구현 가능

❓배열❓
- 반복되는 정보를 처리하기 위해 사용
- for문으로 더욱 편하게 사용
- 잔디테스트용 추가내용 + 추가내용
